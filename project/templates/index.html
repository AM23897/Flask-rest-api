Api Breakdown

Store

  • GET /stores

    Allows you to return a list of all stores from the database in json format

  • GET /store/id

    Allows you to return one store with the specified id from the database in json format

  • POST /store/name

    Allows you to Create a new store into the database with the specified name

  • POST /store/id/delete

    Allows you to delete a store from the database with the specified id. Note: This will require authentication

  • PUT /store/id/update

    Allows you to update a store. For instance the name of the store from the database with the specified id. Note: This will require authentication.

  • GET /stores/param

    Allows you to search for stores from the database with the specified search param.The response is a json object.
